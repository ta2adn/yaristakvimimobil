# 2026 veri paketi – ilk grup

Bu sürüm Formula 1, Formula 2, Formula 3, Formula E, MotoGP, Moto2, Moto3 ve FIA WEC takvimlerini yükler.

Yayımlanmış yarış tarihleri kullanılmıştır. Ayrıntılı seans saati kesin olmayan kayıtlar tüm gün etkinliği ve `STATUS:TENTATIVE` olarak tutulur. Sitede **Saat bekleniyor / Time TBA** görünür; saat uydurulmaz.

Takvimler `data/2026/` klasöründedir. Site sekiz ICS dosyasını paralel olarak okuyup tek takvimde birleştirir.
